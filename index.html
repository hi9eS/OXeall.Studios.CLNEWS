Introducing — OXSync
Official server synchronization system by OXeall Studios.
News
Official
OXSync
▼
OXeall Studios officially introduces OXSync, an official server-side global synchronization system designed as a core infrastructure standard for the studio’s projects. OXSync is built to synchronize server processes, game and system events, as well as user states across servers and game sessions. The system ensures consistent server logic, reduces the impact of network instability, and improves overall predictability of gameplay behavior. The primary objective of OXSync is network stabilization. By optimizing server-side processing and data exchange. Gameplay feels smoother and more responsive — with reduced perceived latency, without modifying actual network metrics. Within the OXeall Studios Status (ALT2) infrastructure, the system is labeled as Global Synchronization and serves as the official source of server environment status. Project support: - City DayX 2: OXSync will be active from the moment of release. Release date: January 2, 2026, 10:45 AM (PST). - Racing Friends: OXSync support is temporarily disabled and will be enabled on December 27. No other studio projects currently use OXSync. OXSync is not a visual optimization or a client-side effect — it is an official server-side system forming the foundation of a stable, scalable, and connected OXeall Studios ecosystem. By OXeall Studios.

Racing Friends Returns — Minor Update with Track Fixes and Map Improvements
A minor update restoring gameplay stability, improving the map, and fixing checkpoint issues.
News
Official
+Minor Update
Racing Friends
▼
Racing Friends officially returns — an important milestone for the project. While the gameplay remains familiar, this update focuses on refining the track and restoring the game to a stable, playable state. - New obstacles have been placed along the track borders to improve race flow and eliminate unintended shortcuts. The spawn building has received several minor adjustments as well, making the starting area cleaner and more consistent with the updated layout. Additionally, checkpoint placement issues have been fixed to ensure that laps are detected correctly and races progress smoothly. This update is not about large-scale additions — it is about bringing the game back online, stabilizing its core systems, and preparing a solid foundation for future improvements and new content.

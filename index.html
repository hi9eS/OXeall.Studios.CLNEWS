Introducing — OXSync
Official server synchronization system by OXeall Studios.
News
Official
OXSync
▾ EXPAND
OXeall Studios officially introduces OXSync — a server-side global synchronization system
designed as a core infrastructure standard for the studio's projects.

OXSync synchronizes server processes, game and system events, and user states across
servers and sessions. The system ensures consistent server logic, reduces network
instability, and improves overall predictability of gameplay behavior.

Project support:
- City DayX 2:     Active from release (Jan 2, 2026, 10:45 AM PST).
- Racing Friends:  OXSync support temporarily disabled, enabled Dec 27.

OXSync is not a visual optimization or a client-side effect — it is an official server-side
system forming the foundation of a stable, scalable, and connected OXeall Studios ecosystem.
